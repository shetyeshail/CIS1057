OPERATING SYSTEM
  os is a special type of software
  program manages all other application software
  it has the responsibility of:
    communicating w/ th euser
    allocating resources(CPU time, memory, etc)
    connecting IO devices w/ running programs
    transferring data b/w main and secondary memory

  to generalize, the os manages processes and resources
  examples: win, osx, linux
  machine usually has a single running os
    many have multiple installed
    can run multiple simultaneously using virtual machines
